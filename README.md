# RQY1102 - Proyecto Edificio El Mirador - Equipo 11

**Sistema de automatización de administración y control de gastos comunes**

---

## Descripción del proyecto

El edificio "El Mirador" (160 departamentos, 20 pisos) enfrenta problemas de gestión manual de gastos comunes, morosidad y falta de transparencia. Este proyecto desarrolla una solución tecnológica que automatiza la administración financiera y operativa, integrando módulos de gestión de residentes, cobranza con WebPay, reportes de morosidad y atención de solicitudes. La arquitectura sigue el modelo 4+1 con enfoque en capas, garantizando seguridad, trazabilidad y usabilidad bajo estándares ISO 25000.

## 📅 Metodología y Control de Versiones

- **Metodología:** Modelo en cascada, con fases secuenciales documentadas en la Carta Gantt.
- **Versionamiento:** Semántico (MAJOR.MINOR.PATCH) aplicado al prototipo y documentos, con repositorio GitHub como fuente de verdad.
- **Commits:** Cada documento fue versionado con mensajes descriptivos (ej. "Agrega diagrama de clases v1.0").
